# Metro Route Finder

This is a Java project developed as part of my college coursework. The application helps users find the shortest route between metro stations using graph-based logic.

## Features

- Display all available metro stations
- Find the shortest path between any two stations
- Simple console-based interface
- Easy to use

## Tools Used

- Java
- IntelliJ IDEA

## How to Run

1. Open the project in IntelliJ IDEA.
2. Run the `Main.java` file.
3. Choose an option from the menu and enter the station names when required.

## Project Structure

- `src` - Contains all Java source files
- `.gitignore` - Git ignore configuration
- `.iml` - IntelliJ project file
